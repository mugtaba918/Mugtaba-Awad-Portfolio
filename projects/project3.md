# SQL Data Analysis & Insights

## Overview
Used SQL to explore relational datasets, combine multiple tables, and produce metrics that support decision-making.

<hr>

## Problem
Operational data often lives across multiple tables and is difficult to analyze without clean joins and aggregations.

<hr>

## Approach
- Joined tables to create meaningful views of the data
- Used aggregations to compute KPIs and summaries
- Checked for anomalies and trends relevant to business questions
- Prepared query outputs for reporting/dashboards

<hr>

## Tools
SQL • Relational Databases • Aggregations • Query Optimization

<hr>

## Results & Impact
- Clear reporting metrics derived from raw tables
- Faster analysis through reusable queries
- Improved transparency for stakeholders using consistent definitions
