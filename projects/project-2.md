# Power BI Stakeholder Dashboard

## Overview
Built an interactive dashboard to help stakeholders track KPIs, trends, and performance quickly without manual spreadsheet work.

<hr>

## Problem
- Static reports slowed decisions
- Manual analysis created inconsistency
- Hard to compare before/after or segment performance

<hr>

## Approach
- Modeled the data for reporting and KPI tracking
- Designed a clean layout with KPI cards and trend visuals
- Added filters/slicers for faster exploration
- Focused on clear storytelling and stakeholder-friendly labeling

<hr>

## Tools
Power BI • DAX • Excel • Data Modeling

<hr>

## Results & Impact
- Faster reporting and clearer visibility into performance
- Reduced manual workload and improved consistency
- Easier executive review through clean, interactive visuals
